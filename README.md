# Lung Cancer Detection
To run the notebook:
1. Clone the repository
```bash
git clone https://github.com/daff00/lung-cancer-classification.git
```
2. Create .venv
```bash
python -m venv .venv
```
3. Activate the .venv 
    - On Windows
    ```bash
    .venv\Scripts\activate.bat
    ```
    - On Mac/Linux
    ```bash
    source .venv/bin/activate
    ```
4. Install the dependencies
```bash
pip install -r requirements.txt
```
